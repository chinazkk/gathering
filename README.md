# Gathering
一款线上活动组团微信小程序

## 微信
![](https://github.com/939682389/gathering/blob/master/WechatIMG12.jpeg)

## 运行环境
- uWSGI、Nginx
- MySQL >=5.1
- python >=3.5

## 功能
- 小组创建
- 活动创建
- 用户聊天
- 消息通知
